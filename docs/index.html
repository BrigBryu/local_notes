Features

Local SQLite Storage

Save notes locally in SQLite database. Your data stays on your device, completely private and secure.

Export & Import

Export individual notes as .txt files or export multiple notes as a .zip archive. Import them back anytime.

No Cloud Required

No account, ads, analytics, or internet access required. Works completely offline for maximum privacy.

Vim Themes

Choose from beautiful themes based on popular Vim color schemes like Gruvbox, Dracula, Solarized, and more.
